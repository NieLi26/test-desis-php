# Sistema de Votacion
el *Sistema de votacion* Desarrollado usando PHP en Backend y MySQL como la base de datos, HTML5, JS y CSS3 en frontend. 
También se usa FormsPree en forma de contacto.


## Tecnologias
- [PHP - v8.1.12](https://www.php.net/docs.php)
- [JS](https://developer.mozilla.org/en-US/docs/Web/javascript)
- [MySQL - v8.0.33](https://www.mysql.com)
- [HTML](https://www.w3schools.com/html/default.asp)
- [CSS](https://www.w3schools.com/css/default.asp)


## Instrucciones para ejecutar el sitio web
- Inicie los módulos Apache y MySQL con el controlador **XAMPP**.
- Abra el **phpMyAdmin** y cree una base de datos **"votacion"**.
- Importe el archivo **votacion.sql** que esta en la carpeta sql.
- Abra la carpeta htdocs en la carpeta XAMPP. Copie y pegue la carpeta **test-desis-php**.
- Abra el navegador, escriba en la url *localhost/test-desis-php*, presione enter  y debe ver la página de index del sitio web.
